📋 Guía Completa — Paso a Paso

✅ Sigue estos pasos en orden. Tiempo total estimado: 1-2 horas la primera vez. Las siguientes instalaciones (para otros clientes): 20 minutos.
1

Crear cuenta en Meta for Developers

Ve a developers.facebook.com → Mis Apps → Crear App → Seleccionar "Empresa" → Continuar.
Luego busca el producto "WhatsApp" y haz clic en "Configurar".

2

Obtener credenciales de prueba

En tu App → WhatsApp → API Setup verás:
• Phone Number ID (cópialo)
• Temporary access token (cópialo — dura 24h para probar)

Para producción necesitarás un System User Token permanente desde Meta Business Suite.

3

Desplegar en Netlify

Descomprime el ZIP y sube la carpeta completa:

npm install # Opción A: Arrastra la carpeta a netlify.com # Opción B: CLI npm install -g netlify-cli netlify login netlify deploy --prod

Netlify te da una URL tipo: https://random-name-123.netlify.app

4

Configurar Variables de Entorno en Netlify

Ve a: Netlify → Tu sitio → Site Settings → Environment Variables

Agrega estas variables una por una:

WHATSAPP_PHONE_NUMBER_ID = tu_phone_number_id WHATSAPP_ACCESS_TOKEN = tu_access_token WEBHOOK_VERIFY_TOKEN = edfix2025secret ADMIN_TOKEN = edfix-admin-2025 BUSINESS_NAME = Nombre del Negocio BUSINESS_HOURS = Lunes a Sabado 9am - 6pm BUSINESS_ADDRESS = Jr. Las Flores 123, Lima BUSINESS_PHONE = +51 999 999 999 BUSINESS_SERVICES = Servicio 1,Servicio 2,Servicio 3 # Opcionales: GEMINI_API_KEY = tu_gemini_key SUPABASE_URL = https://xxx.supabase.co SUPABASE_ANON_KEY = eyJhbGci...

Después de agregar variables → Trigger deploy (redeploy obligatorio).

5

Configurar el Webhook en Meta

En Meta App → WhatsApp → Configuration → Webhook:
• Callback URL: https://tu-sitio.netlify.app/webhook
• Verify token: edfix2025secret (el que pusiste en .env)
• Clic en Verify and save
• Suscribir a: messages

6

Probar el bot (número de prueba Meta)

Meta te da un número de prueba gratuito. Puedes agregarle hasta 5 números de teléfono para testear:
App → WhatsApp → API Setup → "To" → Add phone number → agrega tu número

Luego escribe "Hola" desde tu WhatsApp al número de prueba de Meta. ¡El bot debe responder!

7

Número de producción del cliente

Para que el BOT use el WhatsApp del negocio del cliente:
1. El cliente necesita una cuenta de WhatsApp Business en un número dedicado
2. En Meta → Add phone number → seguir el proceso de verificación
3. El número ya no puede usarse como WhatsApp normal

Alternativa barata: Comprar un chip nuevo (S/5–10) solo para el bot.

8

Supabase — Tabla para leads (Opcional)

Ejecutar este SQL en Supabase → SQL Editor:

CREATE TABLE whatsbot_leads ( id BIGSERIAL PRIMARY KEY, name TEXT, phone TEXT, query TEXT, business_name TEXT, source TEXT DEFAULT 'whatsapp_bot', status TEXT DEFAULT 'new', created_at TIMESTAMPTZ DEFAULT NOW() ); -- Permitir inserciones desde el bot ALTER TABLE whatsbot_leads ENABLE ROW LEVEL SECURITY; CREATE POLICY "insert_leads" ON whatsbot_leads FOR INSERT WITH CHECK (true); CREATE POLICY "read_leads" ON whatsbot_leads FOR SELECT USING (true);
